# Generated Content

`generated-content` es un espacio interno para almacenar contenido de apoyo creado fuera del flujo principal del juego.

## batches

La carpeta `batches` contiene lotes completos de ejercicios (por ejemplo, exportaciones JSON de laboratorio o pruebas de generación).

## exercises

La carpeta `exercises` contiene ejercicios individuales o fragmentos reutilizables para revisión, curación y edición manual.

## Origen del contenido

El contenido guardado en esta zona puede ser creado manualmente o generado con herramientas de IA.

## Alcance

Esta estructura no forma parte del core del juego y se mantiene aislada para no afectar la lógica principal ni el runtime de producción.
